* Hana 提供了异构容器和算法，合并了异构计算和编译期计算。参考文档在[这里](./hana-reference-zh.md)

* Hana 是一个头文件库，利用C++14技术和惯用法编写，不依赖于其它库。头文件文档在[这里](./hana-header-zh.md)
